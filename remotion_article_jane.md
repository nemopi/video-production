# 「完璧」じゃなくて「上出来」を目指す、私たちのための動画生成ハック ——Remotion Agent Skills試用記

正直、タイムラインで「プロンプトだけで動画が作れる」って見たとき、「はいはい、またそれね」って鼻で笑っちゃったのよ。
私たち、こういう「魔法のツール」に何度踊らされてきた？ ポチッとするだけでキラキラした生活が手に入る、みたいな。そんなわけないじゃない？

でもね、結論から言うわ。
**「Remotion Agent Skills」、これは魔法じゃない。もっと泥臭くて、だからこそ愛せる「有能な執事」だったの。**

## 「わからない」を武器にするシステムハック

そもそもRemotionって、コードで動画を作るっていう、聞いただけで蕁麻疹が出そうなシロモノなわけ。エンジニアの皆さんが誇らしげに使う「女の甲冑」ならぬ「男の甲冑」みたいな。

普通なら「私には無理」で回れ右案件。でも、今回のこれは `npx skills add ...` って呪文を一回唱えるだけで、Claude Codeっていう有能な執事が雇えるっていうじゃない。

「へえ、コード書かなくていいんだ？ なら試してやろうじゃないの」

この、**「真正面から技術を習得するのはコスパ悪いけど、使えるものは利用してやろう」という図々しさ。** これこそがネオ中年の処世術よ。

## 執事に「雑用」を投げつける快感

実際にやってみたわよ。「このツイート解説して」ってプロンプト、もとい「命令」を投げたの。

そしたら彼（AI）、文句ひとつ言わずにTypeScriptっていう暗号みたいなコードを生成してきた。
中身を見ると、フレーム数計算だの、補間処理だの、絶対自分でやりたくない計算がギッシリ。

ここで気づいたの。
**「あ、これ私が理解しなくていいやつだ」って。**

私たち、仕事でも家庭でも「全部自分で把握してなきゃ」って自意識の泥沼にハマりがちでしょう？
でもこのツールは、「面倒な計算と実装は僕がやりますから、奥様は『何を作りたいか』だけ指示してください」って言ってるの。

これって、究極の**「責任の外部化」**じゃない？ 最高じゃない？

## バグ？ ああ、愛嬌ってことで

もちろん、完璧じゃないわよ。プレビュー画面が開かなかったり、文字が重なったり。
「おいおい、詰めが甘いぞ」ってツッコミ入れたくなる場面もあった。

でもね、「ここ直して」って言えば、「仰せのままに」って秒で直してくる。
このスピード感。人間相手だとこうはいかない。「なんでそんな言い方するんですか」とか言われて消耗戦になるのがオチ。

AI相手なら、気を使う必要も、機嫌を伺う必要もない。
**私の機嫌を損ねずに、淡々と修正してくれる。**
これからのパートナーに求める条件、これ一択じゃない？

## 結論：ズルして賢く生き抜こう

このツールを使ったからって、私たちが凄腕エンジニアになれるわけじゃない。
でも、「エンジニアみたいな顔して動画を作る」ことはできる。

**「仕組みはよくわかんないけど、できたからヨシ！」**
この「暫定的な着地」を許せることが、大人の余裕ってものでしょ。

学習コスト？ ゼロじゃないけど、「見てればなんとなくわかる」レベル。
誰かの受け売りコードをコピペして冷や汗かくより、AIに書かせて「ふーん」って眺めてる方がよっぽど精神衛生にいいわ。

だから、もしあなたが「動画とか難しそう」って二の足を踏んでるなら、一回これ使ってみて。
完璧じゃなくていい。「ひとまず上出来」な動画が、拍子抜けするくらい簡単に作れるから。

浮いた時間で、マッサージにでも行きましょうよ。私たちは十分頑張ってるんだからさ。
